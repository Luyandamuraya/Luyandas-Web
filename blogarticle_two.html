Published July 2025 · 5 min read

Data Analysis is a fundemental pillar to building the correct insights, but, if done incorrectly it can do harm!
These are some of my lesson's on data anaysis.

Lesson 1: Stakeholder Requirments

Often, a project will have multiple stakeholders with various requirments. This means that each person has a different idea on the output and what they would like to see. To avoid micommunication and ensure the project runs smoothly it's important to communicate effectively.
This can be done through:

  • Prioritising Stakeholders
  • Constantly Recieving Feedback
  • Plan!
  • Creating a plan for communication and implementation
